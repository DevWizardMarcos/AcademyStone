# 📚 Glossário do AcademyStone

## 🎓 O que é um Instrutor?

No AcademyStone, cada instrutor é representado como uma carta colecionável inspirada nos jogos de cartas estratégicos.

Cada carta possui atributos, classe, especialização e habilidades únicas que representam conhecimentos técnicos e experiências reais.

---

## 💠 Atributos

### ⚔️ Ataque

### 🛡️ Defesa
Experiência para lidar com problemas complexos.

### 🧠 Inteligência
Conhecimento técnico acumulado.

### ⚡ Velocidade
Agilidade na entrega de soluções.

### 🤝 Liderança

---

## ✨ Habilidades

Cada instrutor possui uma habilidade especial exclusiva.

### Exemplo

**Componentização Arcana**
> Transforma interfaces complexas em componentes reutilizáveis e escaláveis.

**Debug Supremo**
> Identifica e corrige erros críticos rapidamente.

**Deploy Divino**
> Publica aplicações sem interromper o ambiente de produção.

---

## 🏰 Academia

A Academia é o ambiente principal do sistema.

Nela é possível:

- Visualizar instrutores
- Pesquisar cartas
- Filtrar por classe
- Consultar perfis
- Cadastrar novos instrutores
- Editar informações
- Gerenciar a coleção

---

## 🎮 Inspiração

O projeto foi inspirado nos clássicos jogos de cartas digitais, especialmente Hearthstone, adaptando sua estética e mecânicas para representar instrutores, conhecimento e evolução profissional.
